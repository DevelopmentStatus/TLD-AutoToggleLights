# AutoToggleLights / Auto Turn-off Held Light
Automatically extinguishes the torch/lantern/flashlight that is being held in your hand when passing time or resting.
No more turning them off manually everytime you're about to sleep!

Made for the latest version od The Long Dark 2.51
# Installation
Refer to this site to install MelonLoader
https://tldmods.net/install.html

After that is installed, drag AutoToggleLights.dll into the Mods folder located in the main directory of The Long Dark
